# 🤖 Agentic engineering — eras and patterns

_updated: 2026-06-12_

### AI engineering has moved through three eras: prompt → context → harness

The craft evolved from crafting single messages (Prompt Engineering, 2022) to curating what fits in the context window (Context Engineering, named by Karpathy, ~2024) to bolting a full machine onto the model — Personalisation, Context, Action, Memory, Delegation (Harness Engineering, named by Mitchell Hashimoto, 2025–26). Each era wraps the last: strip the harness and you have a chatbot.
