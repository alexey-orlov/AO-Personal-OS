## Top items to study this session

#### 1. 🟡 Missing article before count nouns / abstract references — 3 instances

You drop *a/the* before singular nouns and named references. In American English these almost always need an article. Highest-leverage fix on this call.

- "based on **product overview** that you had last Friday" → based on **the product overview** you had last Friday
- "they may need to create **certain semantic layer**" → they may need to create **a** semantic layer
- "whether we'd focus on GenAI or even **broader scope**" → or even **a** broader scope

**Drill:** before any singular noun, pause on "the / a / my?" — if none fits, it's probably a rare uncountable. Default is *the* for something already in play, *a* for first mention.

#### 2. 🔴 Run-on / over-complex sentence — 1 instance

One idea got stretched across several stacked clauses; by the end the listener can't hold the thread. Land the main point in ~12 words, then add detail in a second sentence.

- "we can think of how this pod should be expanded **in a way in order to address both of those** because I think the core idea is that not only to do the data engineering work but also enable them at once…" → "We should think about how to **expand this pod to cover both** — the data engineering **and** enabling their agentic use cases."

**Drill:** when you hear yourself say "in a way in order to" or chain a third "because/that", stop and start a new sentence. One breath = one idea.

#### 3. 🔴 Wrong word — *translate* (verb) used as a noun — 1 instance

*Translate* is only a verb. The noun is *translation* (here, a *translation layer*). As said, it briefly stops the listener.

- "a semantic layer that is **this new translate** for their existing database" → a semantic layer that acts as **a new translation layer** for their existing database

**Drill:** noun forms — translate → **translation**, integrate → **integration**, deploy → **deployment**. If it's the *thing* not the *action*, you need the -tion/-ment form.

#### 4. 🟡 Number agreement (subject–verb, demonstrative, plural) — 3 instances

Singular/plural markers drift apart: a plural subject with a singular verb, *this* with a plural noun, a plural meaning with a singular noun.

- "There **was** no insights on that" → There **were** no insights
- "one of the action items was **this** Gero **roles**" → **these** Gero roles (or **the** Gero roles)
- "those **offering** from Big Data cover perfectly" → those **offerings** cover perfectly

**Drill:** check the noun's number first, then make the verb and *this/these* match it. Plural noun → *were / these / -s*.

#### 5. 🟡 Indirect-question inversion — 1 instance

After *what / how / whether*, keep statement word order (subject before verb) — don't invert as in a direct question. You got one right ("ideas on what those roles **might be**") and one wrong.

- "maybe you can share **what was the final point**" → share **what the final point was**

**Drill:** "I asked what **the price was**" (✓), not "what **was the price**." Embed it → verb goes last.

#### 6. 🟡 Wrong preposition in fixed collocation — 2 instances

- "**makes sense for me**" → makes sense **to me**
- "do you have ideas **on** what those roles…" → ideas **about** what those roles…

**Drill:** *makes sense **to** someone*, *agree **with** someone*, *depend **on***. Memorize the verb+preposition as one unit.
