## Top items to study this session

#### 1. 🟡 Missing / incorrect article — 5 instances

You drop or misuse `a/the` in noun phrases, especially in fast delivery. English requires an article on almost every singular countable noun.

- "Thank you for **feedback**" → Thank you for **the** feedback
- "which is quite **significant improvement**" → which is quite **a** significant improvement
- "So **system propose** something" → So **the** system **proposes** something

**Drill:** before any singular noun, ask "a, the, or my/your?" — the answer is almost never "nothing". *the feedback · the system · a significant improvement · a draft.*

#### 2. 🟡 Word order: "talking *here about*" — 3 instances

The particle stays glued to its verb: **talk about X**. Adverbs like *here* go after the whole phrase, not inside it.

- "the use case we are **talking here about**" → the use case we're **talking about here**
- "what we're **talking here about** is applicable" → what we're **talking about** here is applicable

**Drill:** never split "talk about". Say "talking about it here", "talking about that now" — adverb last.

#### 3. 🟡 Non-native verb-noun collocations — 3 instances

The grammar is fine but the verb pairs oddly with the noun; a native speaker picks a different verb.

- "we focus on the KPIs that **defend** the ROI" → that **justify** / **support** the ROI
- "It **turns down** the scheduling cycles from days to minutes" → It **cuts** / **brings down** the scheduling cycles
- "to see that those KPIs are actually **growing up**" → are actually **going up** / **climbing**

**Drill:** *turn down* = reject/lower volume; for numbers use *cut, reduce, bring down*. *grow up* = what children do; metrics *go up*. ROI is *justified/supported*, not defended.

#### 4. 🟡 Verbosity / over-complex sentences — 3 instances

You sometimes state the same idea twice in one sentence or stack near-synonyms, and the listener has to untangle it.

- "the current manual **heavily manual**, poorly scalable, and consequently **suboptimal optimization, workforce optimization** process" → the current manual, poorly scalable, and therefore suboptimal workforce-planning process
- "we envision it might be applicable, what we're talking about is **applicable** to multiple verticals" → this applies to multiple verticals
- "It allows them to enable more extensive OCI usage. **Among their customer base, among their accounts.**" → It drives more OCI consumption across their accounts

**Drill:** after a sentence, check for a word doing the same job twice (*manual heavily manual*, *suboptimal optimization*, *customer base / accounts*). Cut one.

#### 5. 🟡 Indirect question — no inversion — 2 instances

Inside a bigger sentence, a question keeps statement word order (subject before verb) and needs *do*-support.

- "depending on how diverse **are the workflows**" → how diverse **the workflows are**
- "or we kind of **postpone** so far?" → or **do we postpone** for now?

**Drill:** embedded question = statement order. "I know how *it works*" (not "how *works it*"). Add "do we / does it" for yes/no ones.

#### 6. 🟡 No future tense after "if" — 1 instance

In a conditional, the *if*-clause uses present, not *will*.

- "there is one more detailed slide **if we'll need**" → **if we need**

**Drill:** "If X *happens*, Y *will* happen." *will* only in the result half, never after *if*.

#### 7. 🟡 Wrong word / adjective form — 2 instances

- "the **algorithmical** approach" → the **algorithmic** approach
- "5% **always should** go with an explanation" → 5% **should always** come with an explanation

**Drill:** adverb (*always, never, also*) sits **after** the modal: *should always, will never, can also*.

#### 8. 🟡 False friend: "so far" → "for now" — 1 instance

Russian *пока* covers both "so far" (up to now) and "for now" (for the time being); English splits them.

- "or we kind of postpone **so far**?" → or do we postpone **for now**?

**Drill:** *so far* = up to this point (looking back). *for now* = for the time being (looking forward). "So far, so good" vs. "leave it for now".
